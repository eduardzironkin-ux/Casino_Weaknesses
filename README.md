# Casino Weaknesses - Redirect Page

Одностраничный сайт-переходник для Telegram канала Casino Weaknesses.

## Структура проекта

- `index.html` - Основная HTML страница
- `styles.css` - Стили оформления
- `script.js` - JavaScript для редиректа
- `casino-image.jpg` - Изображение казино (нужно добавить)

## Установка

1. Поместите изображение `casino-image.jpg` в корневую папку проекта (или обновите путь к изображению в `index.html`)
2. Откройте `index.html` в браузере

## Настройка

Чтобы изменить URL редиректа, отредактируйте переменную `telegramUrl` в файле `script.js`.

## Особенности

- Темная тема в стиле казино с золотыми акцентами
- Адаптивный дизайн для мобильных устройств
- Плавные анимации и переходы
- Призыв к действию для повышения конверсии
